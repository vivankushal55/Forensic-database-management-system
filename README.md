# Forensic Database Management System

A web-based **Forensic Database Management System (FDMS)** designed to store, manage, and track forensic evidence, crime cases, criminals, and related investigation data in a centralized and secure manner.

This project was developed as a **DBMS Mini Project** to reduce manual paperwork and improve accessibility, collaboration, and data consistency in forensic investigations.
